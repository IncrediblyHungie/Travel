🗺️ Travel Map Configuration Guide

📅 Journey Start Date: September 29, 2025

Maine (Cadillac Mountain) now starts on September 29, 2025

The journey progresses: Maine → New Hampshire → Vermont → New York

🔧 Two View Options:

🎯 Option 1: Limited View (Next 4 Days)

Shows: Only the next 4 destinations starting from September 29

Markers: 4 total (Maine, New Hampshire, Vermont, New York)

🌐 Netlify Environment Variables:

START_DATE_OVERRIDE=2025-09-29
DAYS_TO_SHOW=3
ENABLE_LIMITED_VIEW=true

📝 OR add to HTML head:

<script>
  window.START_DATE_OVERRIDE = '2025-09-29';
  window.DAYS_TO_SHOW = 3;
  window.ENABLE_LIMITED_VIEW = true;
</script>

🌍 Option 2: Full View (All 50 States)

Shows: Complete 50-state journey map

Markers: 50 total (all destinations)

🌐 Netlify Environment Variables:

START_DATE_OVERRIDE=2025-09-29
DAYS_TO_SHOW=3
ENABLE_LIMITED_VIEW=false

📝 OR add to HTML head:

<script>
  window.START_DATE_OVERRIDE = '2025-09-29';
  window.DAYS_TO_SHOW = 3;
  window.ENABLE_LIMITED_VIEW = false;
</script>

🚀 Quick Test

To test full view immediately, add this to the top of index.html:

<script>
  window.ENABLE_LIMITED_VIEW = false;
</script>

This will override the default and show all 50 destinations!

📊 What Changes:

  • Limited View (true): 4 markers, focused map view, route progression 1→2→3→4
  • Full View (false): 50 markers, full USA map, complete journey progression
